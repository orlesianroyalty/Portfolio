Gaia

codeC#
videogame_assetProject Lead
peopleGroup Project

Overview

Gaia is a 2D sidescrolling game in a 3D world, inspired by Paper Mario. The game was originally for a game jam about versatile verbs on a team of only two people but then expanded to a year long project with a larger team of students.


Goal

The goal for this project was to explore versatile verbs. This was acheived with the player's abilities since buttons would have to pressed for different lengths of time or combine different button presses to make it across platforms.


My Role

I helped come up with the concept for Gaia. I was the project leader for this project after the initial game jam. I assigned tasks to my team members. I also hand drew some of the art used such as the flora and Gaia. I also wrote code for some of Gaia's abilities and wrote the code for Fenrir's AI statemachine for the boss battle.

The Forest

codeC#
peopleGroup Project
Hackathon

Overview

The Forest is a horror game in which the player is camping in the forest. Your goal as the player is to escape the forest before your flashlight battery dies. If curiosity gets the better of you, you can attempt to find all of the clues about the forest before your battery dies.

This game was made for the hackathon VandyHacks. We chose to combine Microsoft's Mixer API with Unity to make an interactive game when livestreamed.

The player (the live-streamer) starts up the game and is given an authentication code that viewers will need to submit through Mixer.com/go to have access to the interactivity of the game. From there, the viewer will see a list of buttons mapped like a controller on their channel page on Mixer.com. These buttons will allow the viewer to spawn enemies directly behind the player to try to scare them or leave an in-game note for the live-streamer to read. There are three different types of notes and each type of note has three variations.


Goal

The goal for this project was to combine Unity with the Mixer API to serve as our "hack" for VandyHacks. We wanted to give viewers watching someone play a game the chance to interact with the streamer's gameplay live.


My Role

I programmed the interactivity portion of the game and worked mostly with the Mixer API. I also programmed the note system.

Tigerbook Social Network

codeC++
Academic Project

Overview

This project is an academic project for my software construction class. It was created in C++ in Linux. It allows a user to send a message to another user or to a group as well as read their messages. All messages are stored in files that are tied to the user's account in Auburn's network so the files can be accessed when logged in to any computer.


Goal

The goal for this project was to explore file editing and reading in C++ as well as programming using Linux.


My Role

This was an individual project so I programmed the project in its entirety.
