The Forest

codeC#
peopleGroup Project
Hackathon

Overview

The Forest is a 3D horror game. Your goal as the player is to escape the forest before your flashlight battery dies.

This game was made for the hackathon VandyHacks. We chose to combine Microsoft's Mixer API with Unity to make an interactive game when livestreamed.

The player (live-streamer) is given an authentication code that viewers will need to submit through Mixer.com/go to have access to the interactivity of the game. The viewer will then see a list of buttons on their channel page which allow the viewer to spawn enemies directly behind the player or leave an in-game note.

Goal

The goal for this project was to combine Unity with the Mixer API to serve as our "hack" for VandyHacks. We wanted to give viewers watching someone play a game the chance to interact with the streamer's gameplay live.

My Role

I programmed the interactivity portion of the game and worked mostly with the Mixer API. I also programmed the note system.
