Dracarys

codeC#

Overview

Dracarys is an arcade style game made in Unity. It is based off of Game of Thrones when Daenerys and Drogon burn the Lannister army down.

The player controls Drogon who is flying over the Lannister army. The player can make Drogon shoot fireballs to score points. Drogon only has 3 lives before the game is over.


Goal

The goal for this project was to familarize myself with Unity more since this was my first solo Unity project.


My Role

This was an individual project so I programmed and designed the project in its entirety.
