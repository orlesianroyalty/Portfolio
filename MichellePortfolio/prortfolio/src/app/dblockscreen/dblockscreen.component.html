Overview

This is an app for android made in Android Studio. It was inspired by the Black Door from The Elder Scrolls video game series in which the door asks you a riddle and you must respond with the correct answer to enter.


Goal

The goal for this project was to create a mobile app for my phone using Android Studio.


My Role

This was an individual project so I programmed and designed the project in its entirety.
